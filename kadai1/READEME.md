# 課題 1:画像変換コマンドを作ろう

画像を指定の形式に変換する
変換後の画像ファイルは元の加増と同じディレクトリに配置されます。

## コマンドラインオプション

| オプション | 値の説明                               | デフォルト値 |
| ---------- | -------------------------------------- | ------------ |
| -src       | 変換前の画像形式(jpg,png,gif から選択) | jpg          |
| -dest      | 変換後の画像形式(jpg,png,gif から選択) | png          |

## 使い方

- ディレクトリを指定して実行

```bash
./kadai1 [...options] [directory]
```
