## 実行例
```bash
./main -i .png -o .jpeg .
```

## コマンドライン引数

| オプション | 説明 | デフォルト値 | 対応形式 |
 | --- | --- | --- | --- |
| -i | 変換前の画像フォーマット | .jpeg | .jpeg(.jpg)  .png .gif|
| -o | 変換後の画像フォーマット | .png | .jpeg(.jpg) .png |

※最後の引数で実行ディレクトリを指定する。
