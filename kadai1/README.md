# 画像形式変換コマンド
指定ディレクトリ以下の画像ファイルを変換します。

# 使用方法
imgconvディレクトリに移動して、以下のコマンドを実行する。
`./imgconv -s jpg -d -png [ディレクトリ]`
もしくは、
`go run main.go -s jpg -d png [ディレクトリ]`

# オプション
* `-s` 変換対象画像の形式(デフォルト：jpg)
* `-d` 変換後の画像の形式(デフォルト：png)
